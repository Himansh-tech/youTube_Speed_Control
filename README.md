## YouTube Speed Controller

This extension makes it easier to change the playback speed on YouTube.  
Instead of opening the settings menu every time, you get simple on-screen controls to speed up or slow down any video.

It's helpful for watching tutorials, lectures, or long videos at your own pace.  
Lightweight, quick, and works on all YouTube videos.

**Features**
- Quick speed up / slow down buttons  
- One-click reset to normal speed  
- Clean and simple UI  
- No impact on video quality  

## How to Download the Extension (ZIP File)

I have added the extension as a ZIP file in this repository.  
You can download it by following these steps:

1. Go to the **ZIP file** in this repository.
2. Click on the file to open it.
3. Click the **Download** button.
4. Extract (unzip) the folder on your computer.
5. Open Chrome and go to: `chrome://extensions/`
6. Turn on **Developer Mode** (top right).
7. Click **Load unpacked**.
8. Select the extracted folder.
9. The extension will now appear in your Chrome toolbar.
