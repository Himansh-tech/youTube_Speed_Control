## YouTube Speed Controller

This extension makes it easier to change the playback speed on YouTube.  
Instead of opening the settings menu every time, you get simple on-screen controls to speed up or slow down any video.

It's helpful for watching tutorials, lectures, or long videos at your own pace.  
Lightweight, quick, and works on all YouTube videos.

**Features**
- Quick speed up / slow down buttons  
- One-click reset to normal speed  
- Clean and simple UI  
- No impact on video quality
